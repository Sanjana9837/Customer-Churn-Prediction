# Customer Churn Prediction

## Overview

This machine learning project predicts customer churn using telecom customer data.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Machine Learning Model

* Logistic Regression

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Encoding
5. Model Training
6. Prediction
7. Model Evaluation

## Result

Achieved approximately 80% accuracy.
